== Working with Administrator view

Use *Administrator* view to configure your instance of the {ProductName} ({ProductShortName}) {WebName}. Specifically, the view supports:

* Configuring credentials for private respositories
* Configuring Git, subversuion, and Maven repositories
* Configuring HTTP and HTTPS proxy settings

[NOTE]
====
You must have Adminstrator credentials to work with *Administrator* view.
====
