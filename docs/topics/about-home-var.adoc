= Use of RHAMT_HOME in This Guide

This guide uses the `RHAMT_HOME` replaceable variable to denote the path to your {ProductShortName} installation. The installation directory is the `{ProductDistribution}` directory where you extracted the {ProductShortName} ZIP distribution.

When you encounter `RHAMT_HOME` in this guide, be sure to replace it with the actual path to your {ProductShortName} installation.
