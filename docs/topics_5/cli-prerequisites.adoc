= Prerequisites

{CLIName} has the following prerequisites:

* OpenJDK 1.8, OpenJDK 11, Oracle JDK 1.8, Oracle JDK 11
* 8 GB RAM

{CLIName} on macOS has the following additional prerequisite:

* Set the `maxproc` value to `2048` or greater.
